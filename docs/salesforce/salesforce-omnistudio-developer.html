Exam Domains

1.0 Omnistudio Fundamentals 18%
  • 1.1Describe the Omnistudio solution as it applies to common customer use case scenarios in different industries (telco, energy, healthcare, public sector, etc.)
  • 1.2Determine a project’s needs by evaluating a customer's requirements, assumptions, risks, and constraints
  • 1.3Explain Managed Package Runtime vs. Standard Runtime, Automatic Upgrades, and Manual Intervention Upgrade
  • 1.4Explain Omnistudio license requirements, Permission Set licenses and Permission Sets
  • 1.5Describe the tools required to complete the end-to-end implementation process (SFCLI, VBT, Migration, etc.)
  • 1.6Explain testing, versioning, cloning where applicable through Omnistudio components
2.0 Flexcards 15%
  • 2.1Configure data sources such as SOQL, Data Mapper, Integration Procedure, Custom, etc. and test resulting JSON structure
  • 2.2Construct single and multi-level Flexcards, organizing data using layouts, and repeaters
  • 2.3Bind essential components to construct the Flexcards and configure their properties
  • 2.4Define and implement complex card actions, including launching Omiscripts, firing events, and updating other components
  • 2.5Configure advanced properties for Flexcards (Event Listeners, Session Variables, Public Attributes of Flexcards, multi-language, Omniscript, child Flexcard, custom LWCs)
  • 2.6Customize Flexcard appearance using styling or themes
  • 2.7Implement conditional rendering logic to show/hide elements or alter component appearance based on conditions using states
  • 2.8Embed Flexcards across UI components and different channels
  • 2.9Describe best practices for Flexcards
3.0 Omniscripts 20%
  • 3.1Implement dynamic interactions using various Omniscript elements (Group, Actions, and Input elements to create an Omniscript)
  • 3.2Use child Omniscripts, Flexcards, and custom LWCs for enhanced interactivity and complex UI configurations
  • 3.3Customize Omniscript appearance using styling or themes
  • 3.4Configure advanced properties for Omniscripts (Mult-language, Save Options, Knowledge Articles, Messaging Framework, etc.)
  • 3.5Embed Omniscripts across UI components and different channels
  • 3.6Describe best practices for Omniscripts
4.0 Integration Procedures 15%
  • 4.1Design and configure server side processes to integrate and orchestrate data between systems
  • 4.2Configure Groups, Data Mapper Actions (Extract, Load, Transform), Standard Actions (Remote Actions, HTTP Actions, Response Actions, Sub-Integration Procedures), etc.
  • 4.3Apply conditional logic, loops, and branching within IPs to control the execution path
  • 4.4Implement chainable integration procedures to split long running processes into multiple transactions
  • 4.5Determine and configure caching mechanisms within IPs to maximize performance
  • 4.6Implement robust exception handling using Try/Catch blocks to manage failures and ensure proper rollback
  • 4.7Configure Response Action to return specific data
  • 4.8Describe best practices for Integration Procedures
5.0 Data Mappers 17%
  • 5.1Define and configure Data Mappers to extract, transform, load and map data between source (JSON, XML, Custom Apex class, etc.) and target (JSON, XML, Custom Apex class, sObject, PDF, Docusign, etc.)
  • 5.2Configure and retrieve data from multiple objects with various filter options using Extract Interface
  • 5.3Configure and retrieve data from a single object and related fields using Turbo Extract Interface
  • 5.4Configure and import data to multiple objects using load interface
  • 5.5Configure the Transform interface type to prepare data for downstream use (JSON, etc.) or document generation (PDF, Docusign)
  • 5.6Configure formulas to create or modify values within Data Mappers (concatenate, arithmetic, date formatting, etc.) across interface types (Transform, Extract and Load interface) for downstream use
  • 5.7Map source and target data structures to achieve expected output structure
  • 5.8Describe best practices for Data Mappers
6.0 Troubleshooting and Debugging 15%
  • 6.1Resolve Flexcards display and state issues
  • 6.2Resolve Omniscripts errors or UI not rendering correctly
  • 6.3Resolve Data Mapper output and Data Mapping issues
  • 6.4Resolve integration procedures execution errors
  • 6.5Resolve caching problems
  • 6.6Resolve incorrect permissions and access issues
  • 6.7Resolve missing or broken JSON
  • 6.8Resolve publishing and deployment issues
  • 6.9Use the Debug Console to identify failures and implement resolutions
